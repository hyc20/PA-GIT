G it 사용

1. 계정 정보 등록 
	git config --global user.name 
	git config --global user.email

2.git_prac 폴더를 git이 관리하는 폴더로 지정 
	git으로 관리할 폴더로 이동 
	git init 

3. 기본으로 생성되는 master 브랜치를 main으로 변경하고 
 	git checkout -b main : main브랜치를 생성하고 해당 브랜치로 이동 
	git branch -M main : 기본 브랜치를 main 브랜치로 지정 
	git status : 상태 확인 
4. 파일을 생성하고 add, commit하기 
	git add<파일명> : 파일을 staging area 추가 
			i : 입력모드 esc : 입력모드 빠져나오기 wq : 저장 종료
	git commit -m 'message' : commit을 하고 commit에 대한 간단한 메세지를 남긴다 
	commit 내역은 git log 명령어를 통해 확인 가능

5. 파일을 마지막 commit 상태로 되될리기 
	git restore <파일명> : 파일을 마지막 commit 상태로 rollback 
	git restore --staged <파일명> staged된 파일은 unstaged 상태로 변경 

6. 파일을 원하는 commit 지점으로 되돌리기 
	git reset --soft <commit> : commit 내역을 되돌리고 commit만 취소(staging area에 파일이 남아있다.)
	gir reset --mixed <commit> : commit 내역을 되돌리고 staging까지 취소 
	gir reset --hard <commit> : commt 내역을 되돌리고 파일을 원하는 commit지점으로 rollback
